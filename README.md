# Paint Tool 配布ページ

このrepositoryは、Paint Toolの配布用ページです。

ソースコードは含めず、GitHub Releasesにビルド済みアプリだけを配置します。

## Windowsでインストールする

1. 右側または上部の **Releases** を開く
2. 最新バージョンを開く
3. **Assets** からWindows用インストーラーをダウンロードする
   - `.msi` がある場合はそれを使用してください
   - `setup.exe` がある場合はそれも使用できます
4. ダウンロードしたインストーラーを実行する
5. Windowsの警告が表示された場合は、このアプリを信頼できる場合のみ **詳細情報** -> **実行** を選ぶ

## アップデート

アプリ起動時に新しいバージョンがある場合、Paint Tool内にアップデート通知が表示されます。

通知内の **更新** ボタンからアップデートできます。

## 注意

初期のビルドではコード署名をしていないため、Windows SmartScreenの警告が表示されることがあります。
