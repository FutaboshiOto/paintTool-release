# Paint Tool

Paint Toolは、ローカルで使うシンプルなペイントツールです。

## インストール方法

Paint ToolはGitHub Releasesからダウンロードします。
GitHubのアカウント登録やログインは不要です。

1. このページの右側、またはスマートフォン表示では下の方にある **Releases** を開く
2. 一番上に表示される最新リリースを開く
3. **Assets** を開く
4. 自分のOSに合うファイルをダウンロードする
5. ダウンロードしたファイルを開いてインストールする

注意: `Source code (zip)` や `Source code (tar.gz)` は開発者向けのソースコードです。通常のインストールには使いません。

## Windows

Windowsでは、最新リリースの **Assets** からWindows用インストーラーをダウンロードしてください。

- 迷った場合は `.msi` ファイルを選んでください
- `setup.exe` ファイルがある場合は、それも使用できます
- `latest.json` はアプリ内アップデート用のファイルなので、手動インストールには使いません

ダウンロードした `.msi` または `setup.exe` をダブルクリックし、案内に従ってインストールしてください。

注意: 初期のビルドではコード署名をしていないため、Windows SmartScreenの警告が表示されることがあります。警告が表示された場合は、このアプリを信頼できる場合のみ **詳細情報** -> **実行** を選んでください。

## macOS

macOS向けの配布ファイルがGitHub Releasesにある場合は、最新リリースの **Assets** からmacOS用ファイルをダウンロードしてください。

- Intel Macの場合は `x64` や `intel` と書かれた `.dmg` または `.app.tar.gz` を選びます
- Apple SiliconのMacの場合は `aarch64`、`arm64`、`silicon` と書かれた `.dmg` または `.app.tar.gz` を選びます
- どちらか分からない場合は、画面左上のAppleメニューから **このMacについて** を開き、チップまたはプロセッサを確認します

`.dmg` を開いた場合は、表示された **Paint Tool** を **Applications** にドラッグしてください。

注意: 現在の自動リリースでは、Windows用インストーラーのみを作成しています。macOS向けの配布ファイルがReleasesにない場合は、まだmacOS向けの一般配布前です。

## アップデート

デスクトップ版は、起動時にGitHub Releasesの最新リリースを確認します。
新しいバージョンがある場合は、アプリ内にアップデート通知が表示されます。

利用者は通知内の **更新** ボタンからアップデートできます。
アップデート完了後、アプリは自動的に再起動します。
